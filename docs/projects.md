# Key Projects

## Semi-Dirac spin liquids and frustrated quantum magnetism on the trellis lattice  
**Ph.D. project**

- Fully symmetric PSG classification using Abrikosov fermions  
- Identified 256 *Z₂* and 448 *U(1)* algebraic PSGs  
- Reduced to symmetry-allowed mean-field Ansätze  
- Computed static and dynamical structure factors  

📄 **Publication:**  
- [Phys. Rev. Research 8, 013191 (2026)](https://doi.org/10.1103/xmtj-4kj9)  
- [arXiv:2509.02663](https://arxiv.org/abs/2509.02663)

**Keywords:** Frustrated Magnets, Quantum Spin Liquids, Projective Symmetry Group
